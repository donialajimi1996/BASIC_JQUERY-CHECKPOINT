BASIC JQUERY

We're going to make a program that looks like Microsoft Word. There will be one paragrah and a bunch of buttons. Each button will have an onclick attribute that will link to a different javascript function. These buttons will have unique names and should do the following : -one button to make the paragraph bold -one button to make the paragraph italic -one button to make the paragraph underlined -one dropdown list that will have a bunch of fonts, and when you click on one the paragraph’s font will change to it. - one dropdown list that will have a bunch of numbers (choose whatever you want) and when you click on it the paragraph’s font size changes to that.
